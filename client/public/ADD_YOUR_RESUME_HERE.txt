Put your actual resume file here as "resume.pdf".

The "Download Resume" button in Hero.jsx links to /resume.pdf, which Vite
serves directly from this public/ folder. Once you add resume.pdf here,
the download button will work automatically - no code changes needed.

You also need a second file here: "resume-java.pdf" - this is what the
Download Resume button links to on the /java route (see the resumeFile
field in data/javaResumeData.js). Same idea, just a second PDF for the
Java-focused resume version.
